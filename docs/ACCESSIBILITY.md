# Acessibilidade

O ProtoFácil deve cumprir WCAG 2.1 AA e priorizar idosos codesigners.

## Regras globais

- Botões e controles principais com alvo mínimo de 48x48px.
- Foco visível com outline de 3px.
- Texto base confortável, no mínimo 18px na interface principal.
- Botões com texto visível; ícones nunca ficam sozinhos.
- Nenhuma informação depende apenas de cor.
- Componentes respeitam `prefers-reduced-motion`.
- Mensagens de erro são simples, em português e associadas ao campo.
- Fluxos principais funcionam com teclado.
- Áreas dinâmicas usam `aria-live` com moderação.

## Editor

O editor usa SVG nativo com alternativas por teclado e anúncios textuais. Arrastar com mouse pode existir, mas nunca será a única forma de executar uma ação.
